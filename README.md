# key mappings   

| 快捷键   | 功能 
| ---------| ------------- |
|`<leader>z` | 挂起`<c>z`| 
|`<leader>q` | 保存退出`:wq<CR>` (or `:x<CR>`)| 
|`<leader>s` | 保存`:w`|
|`<leader>/`  | 取消高亮|
|`<leader>h` | 跳转`<c-w>h`| 
|`<leader>j` | 跳转`<c-w>j`|
|`<leader>k` | 跳转`<c-w>k`|
|`<leader>f` | 两字符查找，来自`easymotion`插件|
|`<leader>d` | 行查找，来自`easymotion`插件|
|`<leader>cc`| 行注释|
|`<leader>cu`| 解注释|
